# homebrew tap

```
$ brew tap dongri/tap
```

## coins

```
$ brew install dongri/tap/coins
```

or

```
$ brew tap dongri/tap
$ brew install coins
```

## GitY

A modern Git GUI client for macOS

```
$ brew install --cask dongri/tap/gity
```

or

```
$ brew tap dongri/tap
$ brew install --cask gity
```

## CmdSwitch

macOS menu bar app to switch input sources with Command key

```
$ brew install --cask dongri/tap/cmd-switch
```

or

```
$ brew tap dongri/tap
$ brew install --cask cmd-switch
```
